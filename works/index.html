Learning Management System

Learning Management System

This system was created for a university and to enable professors and students to have a communication strongly and easily. Wherever they are, they can access to this system from their PC or phone, check a latest information and take some actions immediately, that must help students to study efficiently.

We narrowed down functions and implement essential things to use more comfortably by researching with university.

It allow professors to:

  • Plan and make class schedule.
  • Distribute the documents, quizzes and assignments to students.
  • Mark quizzes and report.
  • Inform students something about class.

It also students to:

  • Make and manage personal schedule.
  • Take quizzes and submit report.
  • Post on a message board to ask questions or give opinions.
  • Check the learning state of themselves and comments by professors.

This was the project that I had the responsibility of design and implementation as a Full-Stack Developer at CMC Solutions Inc., a Japanese IT company where I worked for over 3 years.

HTML/CSS | JavaScript | Unix | PHP | PostgreSQL

MEAMO

MEAMO

This Android application was the one that I created with Java and Google APIs in my class during studying at Cornerstone International Community College of Canada.

When I go to restaurant, bar or cafe, I try to order the food and the drink that I ordered last time, but I often forget a name of them even if those were really good. Sometimes even the location…

That’s why I decided to create this application to avoid that situation.
I can make a memo about the restaurant with:

  • Category (e.g. Bar, Dinner, Cafe, FastFood)
  • Name
  • Rating for each category (e.g. Food, Drink, CP which means Cost Performance)
  • Location
  • Text memo
  • Visit date
  • Photo

It could help someone like me to order their favorite food!

Java | Google APIs
LR

LR

I created this iOS application with Swift and SpriteKit in my class during studying at Cornerstone International Community College of Canada.

It is king of puzzle game and its rule is simple.

  • 2 characters ("L" and "R") are falling from top to down, and those characters have 2 colors (Red and Black).
  • Black : Swipe to the direction that the character represents. (e.g., If “R” is falling down, swipe RIGHT)
  • Red : Swipe to the direction which lane the character is on.

If you swipe to wrong direction, or character disappear before you do, it's GAME OVER!

It might be more difficult that you think…

Swift | SpriteKit
